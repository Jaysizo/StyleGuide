Layout

We have implemented a flexbox-based responsive grid system, adopting it to our class name syntax. The grid works by adding classes that align to specific breakpoints to the grid cells or containers. When those classes match their corresponding breakpoint the grids adjust accordingly.

For example, the grid below consists of two containers, both share the same CSS class names. The first container holds 2 columns. The second container holds 3 columns. Each container has two "media classes" assigned to it:

  1. .g-B--full
  2. .g-ML--fit

The class names can be parsed as follows:

  • g stands for Grid.
  • B and ML are breakpoint sizes that correspond to base (default) and 48em/768px, respectively.
  • full will make the columns full width.
  • fit will make the columns fit inside their container.

Resize your browser window to see how the grid behaves.

Grid classes

The full explanation of each grid class and what it controls is below:

Basic

l-grid
Identifies an element that will act as the grid container.
g-cell
Identifies an element as a cell of the grid.

Modifiers

g-*--gutters
Creates a gutter, to the left, of each grid cell of 0.5rem (8px)
g-*--guttersLarge
Creates a larger gutter of 1rem (16px)
g-*--gutters2x
Creates a larger gutter of 2rem (32px)
g-*--gutters3x
Creates a very large gutter of 3rem (48px)

Cells

g-*--full
Creates full width cells at the specified breakpoint.
g-*--fit
Creates cells that flex to be equal size with the other cells in the grid.
g-*--1of2
A cell that is 50% of the total width of the grid.
g-*--1of3
A cell that is 33% of the total width of the grid.
g-*--1of4
A cell that is 25% of the total width of the grid.

More

For more on the Flexbox grid see the Solved by Flexbox documentation.

Buttons

All buttons are text and/or an icon enclosed in a rectangular shape. NCARB blue is used as the primary call-to-action button and secondary buttons can be light yellow, dark grey, green or beige.

Usage:

The .btn class will style a default button.

Adding the .block class to a .btn will make the button a block element with 100% width.

Link button
.btn-primary 
                  
  <button class="btn-primary" type="button">
    Button
  </button>
  <a href="#form_elements" class="btn-primary continue">
    Link button
  </a>

Usage:

The .btn-primary class will create a primary button. A primary button should be reserved for the main call to action for an entire page regardless of whether the button is found within a form, for selecting an item from a collection, applying a filter, or some other action.

Adding the .continue class to any .btn adds a right facing arrow to the button. It should be used when selecting the button would continue to the next step of a process.

Imagery

Both illustration and photography can live within the same NCARB product such as features, topics and promotions. Choose images that express personal relevance.

Ideally, assets should retain a 1:1 or a 4:3 aspect ratio. The aspect ratio of a photographic image is the proportional relationship between its width and its height. It is expressed in width : height. Photographic images with consistent aspect ratios can easily be scaled and repurposed in various responsive layouts within NCARB.org and NCARB digital products. However, there are exceptions to features, topics and promotional images.
