Layout

Flex-box layout with bootstrap compatability, with bootstrap-tech fall-back for IE9-.

Grid classes

Basics

.flex-container
Required: identifies an element that will act as the grid container.
.flex-item
Required: identifies an element as a item of the grid.
breakpoints sm, md, lg
optionally inserted after the flex prefix in the classes below specify layout breakpoints

Container Classes

Fitting

.flex-2
Expects 2 item-units, evenly spacing 2 normal items; this unbreakpointed class is required for IE9- support. Caveats: For modern browsers, this class is unnecessary and items are evenly spaced across one row of the container; for IE9-, each item will be at 50% width; 3+ items wrapping to extra rows.
.flex-md-4
Spaces items 4 per row when the browser is regular desktop size or greater.
.flex(-sm|-md|-lg)?-[2-12]
Pattern

Wrapping

.flex-1of2
Spaces items at 50%; not required if using similar classes on all flex-items
.flex-md-1of3
Spaces items at 33% when the browser is regular desktop size or greater
.flex(-sm|-md|-lg)?-1of(2|3|4|6|8|12)
Pattern

Gutters

.flex-gutters
Creates a 0.5rem (8px) gutter between each item
.flex-sm-gutters-wide
1rem (16px) for devices tablet-sized or greater
.flex-gutters-wider
2rem (32px)
.flex-gutters-widest
3rem (48px)
.flex(-sm|-md|-lg)?-gutters(-wide(r|st)?)?
Pattern

Items

Fitting

Used in conjuction with a container fitting class like .flex-3

.flex-item only
An item that takes one unit of space indicated by the container number class.
.flex-3
An item that takes three units of space of a normal item.
.flex-sm-3
An item that takes three units of space of a normal item on a device tablet-size or larger.
.flex-full
An exception to fitting: item 100% the width of the container on its own row.
.flex(-sm|-md|-lg)?-([2-12]|full)
Pattern

Wrapping

Used in conjuction with a container wrapping class like .flex-1of2

.flex-1of2
An item 50% the width of the container.
.flex-sm-2of3
An item 66.66% the width of the container on a device tablet-size or larger.
.flex-md-full
An item 100% the width of the container on a browser of a regular screen or larger.
.flex(-sm|-md|-lg)?-(full|1of(2|3|4|6|8|12)|{n-1}of(3|4|6|8|12))
Pattern
