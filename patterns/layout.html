Layout

flex-box layout with bootstrap compatability, with bootstrap-tech fall-back for IE9.

Grid classes

Basics

.flex-container
Required: identifies an element that will act as the grid container.
.flex-item
Required: identifies an element as a item of the grid.
breakpoints sm, md, lg
optionally inserted after the flex prefix in the classes below specify layout breakpoints

Container Classes

.flex-2
Spaces items at 50%; not required if using XofN classes on all flex-items
.flex-md-3
Spaces items at 33% when the browser is regular desktop size or greater
.flex(-sm|-md|-lg)?-[2-12]
Pattern

Gutters

.flex-gutters
Creates a 0.5rem (8px) gutter between each item
.flex-sm-gutters-wide
1rem (16px) for devices tablet-sized or greater
.flex-gutters-wider
2rem (32px)
.flex-gutters-widest
3rem (48px)
.flex(-sm|-md|-lg)?-gutters(-wide(r|st)?)?
Pattern

Items

Used in conjuction with a container class like .flex-4; such a container class is not needed if every item is specified with an XofN class.

.flex-4 > just .flex-item
An item the whole width of the container.
.flex-4 > .flex-item
An item one part the width of the container; here it’s 25%.
.flex-4 > .flex-2
An item 50% the width of the container.
.flex-sm-2of3
An item 66.66% the width of the container on a device tablet-size or larger.
.flex-sm-2 .flex-md-full
An item two units wide on a device tablet-size and 100% the width of the container when the browser is regular desktop size or greater
.flex(-sm|-md|-lg)?-(full|([1-11]of)?[1-12])
Pattern
